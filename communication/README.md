# Communication

Skills for communicating clearly and having the conversations that matter — structuring written messages for impact, escalating problems without creating drama, and navigating difficult face-to-face conversations with directness and care.

## Skills

| Skill | Purpose |
|-------|---------|
| `bluf-communication` | Improve written communication using the BLUF framework — make messages clearer, more direct, and easier to act on. |
| `difficult-conversations` | Structure and prepare for high-stakes conversations you've been avoiding — feedback, underperformance, conflict. |
| `escalate-without-drama` | Helps you escalate a problem — something off track, a blocked dependency, unclear ownership, or a team that isn't moving — without sounding political, blamey, or alarmist. |
| `reframe-for-execs` | Compress and reframe a detailed or operational draft into a crisp executive-ready message. |
